⚠️ Backend server not running

The assistant needs a local server to work.

If you've already set up the background service, the server should start automatically when you log into your Mac. If it's still not working, make sure your project folder isn't in ~/Desktop or ~/Documents — macOS blocks background services from those locations.

Otherwise, open a terminal and run:

cd SciKick && ./start.sh

Then click Retry below or the ⟳ button.
